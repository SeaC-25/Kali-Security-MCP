# 🖥️ 本机依赖环境安装指南 (Windows)

## 📋 概述

本文档详细说明如何在Windows本机上配置KaliMCP所需的依赖环境，包括Python环境、逆向分析工具和PWN工具等。

## 🔧 基础环境要求

### 系统要求
- **操作系统**: Windows 10/11 (64位)
- **内存**: 至少8GB RAM (推荐16GB+)
- **存储**: 至少10GB可用空间
- **网络**: 能够访问Kali Linux服务器

## 🐍 Python环境配置

### 1. Python 3.8+ 安装
```powershell
# 推荐使用官方Python安装包
# 下载地址: https://www.python.org/downloads/
# 安装时勾选 "Add Python to PATH"
```

### 2. 核心Python依赖
```bash
# 升级pip
python -m pip install --upgrade pip

# MCP核心依赖
pip install fastmcp requests socketio-client

# Web工具依赖
pip install beautifulsoup4 lxml

# PWN工具依赖 (PwnPasi)
pip install pwntools>=4.9.0
pip install LibcSearcher>=1.1.5
pip install ropper>=1.13.5

# 逆向分析依赖
pip install pyelftools
pip install pefile
pip install capstone
```

### 3. 可选Python依赖
```bash
# IDA Pro MCP支持
pip install https://github.com/mrexodia/ida-pro-mcp/archive/refs/heads/main.zip

# 加密分析工具
pip install pycrypto pycryptodome

# 网络分析工具
pip install scapy
```

## ⚡ PWN工具环境

### 1. WSL2 (Windows Subsystem for Linux)
```powershell
# 启用WSL2 (管理员权限)
dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart
dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart

# 重启后设置WSL2为默认
wsl --set-default-version 2

# 安装Ubuntu
wsl --install -d Ubuntu
```

### 2. PwnPasi工具验证
```bash
# 进入pwnpasi目录
cd F:\kali\MCP-Kali-Server-main\pwnpasi

# 测试PwnPasi
python pwnpasi.py --help

# 测试样本二进制文件
python pwnpasi.py -l Challenge/level3_x64
```

## ⚠️ 常见问题和解决方案

### 1. Python包安装失败
```bash
# 使用国内镜像源
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple package_name

# 升级pip
python -m pip install --upgrade pip

# 清理pip缓存
pip cache purge
```

### 2. 权限问题
```powershell
# 以管理员权限运行命令提示符
# 右键 -> "以管理员身份运行"

# 或使用PowerShell管理员模式
Start-Process powershell -Verb runAs
```

### 3. 防火墙和网络问题
```powershell
# 检查Windows防火墙设置
# 控制面板 -> 系统和安全 -> Windows Defender 防火墙

# 添加Python和相关工具到防火墙例外
```

### 4. WSL2相关问题
```bash
# 检查WSL2状态
wsl --status

# 重启WSL2
wsl --shutdown
wsl

# 更新WSL2
wsl --update
```

### 5. pwntools在Windows上的问题
```bash
# 如果pwntools在Windows上有问题，使用WSL2
wsl
sudo apt update
sudo apt install python3-pip
pip3 install pwntools
```

## 🚀 性能优化建议

### 1. 系统优化
- **SSD存储**: 使用SSD提高文件I/O性能
- **内存**: 16GB+内存用于大型二进制分析
- **CPU**: 多核处理器用于并发扫描

### 2. Python优化
```bash
# 使用虚拟环境隔离依赖
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

### 3. 工具配置优化
```bash
# Radare2配置优化
echo 'e asm.syntax = intel' >> ~/.radare2rc
echo 'e scr.color = true' >> ~/.radare2rc
```
